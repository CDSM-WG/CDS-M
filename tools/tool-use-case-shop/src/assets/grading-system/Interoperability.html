Grading System Standards

Interoperability Scores 

A
- Clearly described. The information is used as intended and described in the standards documentation, so a clear mapping to other standards/domains can be made.

B
- Filtered. The format is not altered, but the data is filtered (e.g. only used to publish information about a subset of the original, described subset in the standard).

C
– Altered data. The format is altered, and fields are misused, not provided, or left blank

D
- Mandatory altered data. As C, but mandatory fields are manipulated

E
- Others. By lack of a clear specification, this cannot be reused OR a combination of B, C, or D.
